# EducateBuddy – IPD Prototype

This is the IPD prototype for EducateBuddy 

## What’s included
- Login / Sign up with email verification (Firebase Auth)
- Create / Join Groups (Firestore)
- Group chat messages (Firestore)

## Firebase setup (for review)
This prototype uses Firebase:
- Authentication: Email/Password
- Firestore: groups, memberships, messages collections

To run successfully, the Firebase project must have:
1. Email/Password authentication enabled
2. Firestore database created (in test mode or with suitable rules)

## How to run
1. Download the repo
2. Open the folder in VS Code
3. Run with a local server (not by double-clicking index.html)
   - VS Code extension: “Live Server”
4. Open the page and test sign up / login
